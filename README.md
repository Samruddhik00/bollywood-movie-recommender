# 🎬 Bollywood Movie Recommendation System

A content-based movie recommender built using Python and Streamlit.

## 🚀 Features

* Recommend movies based on similarity
* Filter by genre, actor, director
* View trending and top-rated movies
* Interactive dashboard with visualizations
* Movie comparison with similarity score

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn (TF-IDF, Cosine Similarity)
* Streamlit
* Matplotlib, Seaborn

## ▶️ How to Run

```bash
pip install -r requirements.txt
python generate_data.py
streamlit run app.py
```

## 📌 Learning Outcome

Built this project while learning data science concepts like:

* Data preprocessing
* Feature engineering
* Content-based recommendation systems
